Как получить рабочую программу? 
Необходимо скачать файлы с данного репозитория и перейти в папку PROGRAM. Далее вручную либо через контекстное меню распаковать архив bilet.zip РЯДОМ с PDD_EXAM.exe
В конечном итоге в папке PROGRAM должны находиться файл PDD_EXAM.exe И ПАПКА bilet!!!

Для работы программы необходим .NET Framework V4.7.2 или выше (не входят в комплект поставки)
